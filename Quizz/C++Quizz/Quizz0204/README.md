## 3. 类和结构体的区别

在C++出现的时候为了兼容C语言, 保留了结构体的概念, 但对结构体做了扩展, C++中的结构体不仅可以包含数据成员, 还可以包含方法成员.

结构体和类除了默认的成员访问权限不同之外, 没有其他区别, 但还是要用class定义,因为class是专门为了面向对象设计的.class默认是private, struct默认是public.
